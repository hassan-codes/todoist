<h1>Todoist</h1>
<p>
    Todoist is a simple CRUD application built with laravel. Todoist employs demonstrates the use of routing, controllers and models in the MVC architecture.
</p>

<h2>Installation</h2>
Run * php artisan serve * in the working directory
